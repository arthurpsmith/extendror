ROR Departmental Extension - Demo

This is a demonstration of a proposed extension of the ROR research organization identifiers, to link institutions with their sub-units in any hierarchy, whether or not the particular sub-unit has its own ROR identifier.

  • Information on the ROR extension project
  • Github site for extension files
  • Github repository with the code for this demonstration site
